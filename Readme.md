# 🚀 Team Portfolio Manager

A simple and efficient **Python-based Team Portfolio Manager** that helps organize team member information, manage portfolios, and perform common management operations through an easy-to-use interface.

---

## 📖 Overview

Team Portfolio Manager is designed to simplify the process of managing a team's portfolio. It allows users to store, update, search, and organize team member details in one place, making collaboration more efficient.

---

## ✨ Features

- ➕ Add new team members
- ✏️ Edit existing member information
- ❌ Delete member records
- 🔍 Search members instantly
- 📋 Display all team members
- 🔄 Refresh data automatically
- 💾 Save portfolio data locally
- 🖥️ Simple and user-friendly interface

---

## 🛠️ Technologies Used

- Python 3.x
- Tkinter (GUI)
- CSV File Handling
- Object-Oriented Programming
